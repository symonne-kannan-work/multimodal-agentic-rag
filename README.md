# Agentic RAG

An AI-powered Agentic Retrieval-Augmented Generation (RAG) application that combines LLMs, semantic search, and agent workflows to answer questions from custom knowledge sources.

> 🚧 **Project Status:** Work in Progress

---

## Features

- Agentic RAG pipeline
- Semantic document search
- LLM-powered question answering
- FastAPI backend
- Streamlit frontend
- LangGraph workflow orchestration
- Gemini embeddings
- Qdrant vector database
- FlashRank reranking
- Guardrails & observability

---

## Tech Stack

- Python 3.11
- FastAPI
- Streamlit
- LangChain
- LangGraph
- Google Gemini
- Qdrant
- FlashRank
- NVIDIA NeMo Guardrails
- LangSmith
